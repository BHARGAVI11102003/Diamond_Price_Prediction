# Diamond_Price_Prediction using ML
Description:

This project predicts diamond prices using regression models and groups diamonds into market segments using KMeans clustering based on physical and quality attributes.

Features:
Data preprocessing and cleaning
Feature encoding (cut, color, clarity)
Regression models for price prediction
KMeans clustering for segmentation
Model evaluation using MAE, RMSE, R²
Model persistence using pickle

Models Used:
Linear Regression
Decision Tree Regressor
Random Forest Regressor
KMeans Clustering

Evaluation Metrics:
MAE
RMSE
R² Score
Clustering inertia

Key Learning:
Feature consistency between training and deployment
Handling categorical encoding
Difference between regression and clustering
Model saving and loading using pickle
